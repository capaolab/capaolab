# Controle de Versão - Git

## Convenção para menssagem de commits

### Estrutura da menssagem
- [branch n°]:feat: Nova funcionalidade

**Exemplo**:
```bash
103:feat: Nova funcionalidade
```
### Tipos adotados
- feat: Nova funcionalidade
- fix: Correção de bug
- docs: Documentação
- style: Alterações que não afetam código 
- refactor: Refatoração de código
- test: Adição de testes
- chore: Alterações no  build ou ferramentas auxiliares
